# githubapis
Check activity on accounts using github api

Features

Fetch recent public activity for a GitHub user

View events such as:

Commits

Pull requests

Issues

Forks and stars

Optional authentication using a GitHub Personal Access Token (PAT)

Handles GitHub API rate limits gracefully

Easy to extend for additional GitHub endpoints

Tech Stack

Language: 

GitHub REST API v3

HTTP client library (e.g. requests, axios, fetch)
